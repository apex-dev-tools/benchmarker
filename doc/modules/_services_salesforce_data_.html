bulkInsert

  • bulkInsert(connection: SalesforceConnection, properties: { csvPath: string; externalIdPrefix: string; nItems: number; objectApiName: string; parentExternalIds?: string[] }, chunkSize?: number): Promise<string[]>
    • Defined in src/services/salesforce/data.ts:107

    Creates SObject records in bulk from a csv file

    example
    //Creates 1000 Opportunities with 2000 OpportunityLineItems each
let createdOppExternalIDs:string[] = [];
const opportunityExternalIDs: string[] = await bulkInsert(connection, {nItems: 1000, csvPath: 'scenarios/product/generic-csv-data/opportunities.csv', objectApiName: 'Opportunity', externalIdPrefix: 'Opp__'}, 200);
createdOppExternalIDs = createdOppExternalIDs.concat(opportunityExternalIDs);
await bulkInsert(connection, {nItems: 2000, parentExternalIds: opportunityExternalIDs, csvPath: 'scenarios/product/generic-csv-data/opportunitiesLines.csv', objectApiName: 'OpportunityLineItem', externalIdPrefix: ''}, 300);

    Parameters

    • connection: SalesforceConnection

      object to handle the connection to a Salesforce Org

    • properties: { csvPath: string; externalIdPrefix: string; nItems: number; objectApiName: string; parentExternalIds?: string[] }

      object that wraps the configuration to create the objects in bulk

      • csvPath: string
      • externalIdPrefix: string
      • nItems: number
      • objectApiName: string
      • Optional parentExternalIds?: string[]
    • Default value chunkSize: number = BULK_API_DEFAULT_CHUNK_SIZE

      number of records to be created in each batch, by default is 2000

    Returns Promise<string[]>
